About

The Project

Tasked by Oklahoma State University, the project definition was given to us as the following:

“A system that snaps a picture and stores the picture in memory securely. A good system will enable a security protocol, hopefully with military-grade encryption, so that any system cannot possibly be compromised by a user or thief. The picture should also be able to be recalled at a later time.”

The Goal

Our goal is to produce a secure camera system that can capture photographs, store them in an off-site server securely, and allow only the intended user to access and retrieve the files at their convenience
